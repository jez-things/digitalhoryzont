Digital horizont might help you mine bitcoins

Digital horizon is software for monitoring data from environment: temperature, humidity or light. It also can monitor hashrate or performance of mining hardware. It is intedned to be installed on rasberry pi as a collection node and linux server for running miunin software.
At a glance its set of python scripts for collecting data localy from phisical sensors or bitcoin miners running cgminer and pushing it into munin hosted on external server.

Instalation

To monitor temperature and hashrate of your bitcoin mine with digital horizont on your own you need:

  • rasberry pi model B with raspian
  • digital eviroment sensor like DS18B20 (temp) or DHT11 (humidity)
  • 2x resistor 4k7
  • protoboard + small cables
  • working instalation of munin software on external server
  • working cgminer for example on antminer.
